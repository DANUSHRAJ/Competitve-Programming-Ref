Hills Numbers:
Hills numbers are ODD lengthed numbers.

Let us imagine for a nine digit number, the indidividual digits are represented from d1 to d9
d1 d2 d3 d4 d5 d6 d7 d8 d9

  d2  d4  d6  d8
d1  d3  d5  d7  d9
All even positioned digits will be larger than their adjacent digits

d2 > d1, d3
d4 > d3, d5
d6 > d5, d7
d8 > d7, d9

Ex: 362317680 is a Hills number
 6 3 7 8
3 2 1 6 0

36241 is a 5 digit Hills number

 6 4
3 2 1

Given a number check if it is a Hills Number and print

"36412 is a Hills Number" or "56471023 is Not a Hills Number"

Input

Output

201 is not a Hills Number

Testcase 4
Input

Output
123456789 is not a Hills Number

Testcase 5
Input

Output
987654321 is not a Hills Number



Prime Number by Interlacing
~~~~~~~~~~~~~~~~~~~~~~~~~~~
Write a 'C' program to take two integers, interlace as per the description and check if the interlaced number is a prime number.

Output : %d and %d Interlaced is %d is a Prime Number
         %d and %d Interlaced is %d is Not a Prime Number
Example : 2133 9127
Interlaced : 29113327
Prime Check : Prime Number 

Output : 2133 and 9127 Interlaced is 29113327 is a Prime Number
